Todo web application, with multiple lists
# Tools
- Reactjs
- Reactstrap
- CSS
- HTML
- Bootstrap

# How to run
- install nodejs
- run command
  ```npm
  npm install
  ```
- run command
  ```npm
  npm start
  ```

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
